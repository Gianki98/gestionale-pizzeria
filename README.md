# 🍕 Gestionale Cassa — Pizzeria Ximenes

Applicazione gestionale professionale per la ristorazione, 
sviluppata per ottimizzare i processi operativi di un'attività familiare.
Supporta più dispositivi connessi in simultanea con sincronizzazione real-time.

## 👨‍💻 Sviluppatore
* Carlo Ganci

## 🚀 Caratteristiche principali
* Interfaccia touchscreen ottimizzata per uso quotidiano
* Sincronizzazione real-time tra più dispositivi (cassa, cucina, sala)
* Funzionamento offline grazie a PWA e Service Worker
* Gestione completa ordini con tavoli, asporto e delivery
* Anagrafica tavoli con stati e ordini attivi
* Notifiche push per nuovi ordini
* API REST per comunicazione client-server
* Database relazionale con persistenza degli ordini

## 🛠️ Tecnologie utilizzate
* **React** — interfaccia utente dinamica
* **HTML5 / CSS3** — struttura e stile responsive
* **Node.js / Express.js** — backend e API REST
* **SQLite** — database relazionale lato server
* **Socket.io** — comunicazione WebSocket real-time
* **PWA / Service Worker** — funzionamento offline e cache
* **GitHub** — version control

## ▶️ Come avviarlo
```bash
npm install
node server.js
```
Apri il browser su `http://localhost:3000`
